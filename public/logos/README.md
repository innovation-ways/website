# Company logos — experience map

Drop the official company logos here. Until a file exists, the map and cards fall
back to a clean text wordmark (no broken images), so the site works as-is.

Expected files (exact names — referenced from `src/data/experience.ts`):

| File                | Company      |
| ------------------- | ------------ |
| `sunrise.svg`       | Sunrise      |
| `orange.svg`        | Orange       |
| `vodafone.svg`      | Vodafone     |
| `bnp-paribas.svg`   | BNP Paribas  |
| `axa.svg`           | AXA          |

**Format.** Prefer **SVG** (crisp at any size). PNG with transparent background is
fine too — if you use PNG, update the path in `src/data/experience.ts`.

**Look.** A simple, mono or full-colour mark on a transparent background reads best
inside the circular pin (~40 px) and the card chip. Avoid logos with a baked-in
white/colour box.

> Note: these are third-party trademarks shown as the founder's employment history
> (the same companies listed publicly on LinkedIn). They are intentionally **not**
> committed by the tooling — add the assets you're comfortable using.
